npm -v

IF YOU DONT HAVE ANGULAR A.K.A NPM INSTALLED, INSTALL IT IN YOUR WOKRING DIRECTORY USING =>
npm install -g @angular/cli

node -v

IF YOU DONT HAVE node js INSTALLED, INSTALL IT FROM =>
https://nodejs.org/en

ng serve
